# Mandelbrot set

Generate and draw the Mandelbrot set.

## Usage

```bash
mandelbrot-set [OPTIONS]

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

OPTIONS:
    -o, --output <output_name> 
```
